## 双精英协同进化遗传算法
### 改进内容：
- 采用两个精英个体，分别按照不同的选择策略完成各自的进化
- 子种群1的进化主要是加快算法的收敛速度，引入水平集概念自动地对种群进行分割，进而通过协作交叉操作避免相似个体间的无效交叉
- 子种群2的主要进化目标是提高种群的多样性，并加入随机种群，通过分阶段调节随机种群的比例参数，更好地平衡算法的勘探和开采操作，避免早熟收敛或者收敛速度慢等问题的产生
